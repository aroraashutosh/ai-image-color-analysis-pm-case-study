# AI Image Color Analysis Tool

AI-assisted image color analysis prototype that classifies image pixels into 11 predefined color categories and generates color distribution charts.

## Features

* Pixel-level image analysis
* Image downsampling for faster and cleaner color detection
* Euclidean distance-based color classification
* Percentage color distribution calculation
* Pie chart visualization

## Tech Stack

* Python
* NumPy
* Pillow (PIL)
* Matplotlib

## Color Categories

White, Black, Red, Green, Yellow, Blue, Brown, Purple, Pink, Orange, Gray

## How It Works

1. Upload an image
2. Image is downsampled
3. Pixels are analyzed
4. Euclidean distance is used to find the nearest color category
5. Color percentages are calculated
6. A visualization chart is generated

## Project Purpose

Built as an AI Product Management case study to demonstrate product thinking, rapid prototyping, AI-assisted development, and data visualization.
